## 设置快捷键

方式：

1. 打开快捷键设置 `cmd+k, cmd_s`
2. 搜索对应的功能关键词


例子：

1. 执行python的快捷键：command + f5 如果不起作用就去设置这个快捷键，如果有重复就去调整。

    * cmd+k， cmd+s 打开快捷键
    * 搜索 Python: Run Python File in Terminal
    * 点加号，进行设置
